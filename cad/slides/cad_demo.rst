INSPIRE Cadaster Workspace
---------------------------------
1. Open Deegree
2. Go to workspaces
3. Import deegree-workspace-inspire.
4. Start the workspace - you will see : Active workspace: deegree-workspace-inspire
5. Get feature request  numberOfFeatures="15000"
